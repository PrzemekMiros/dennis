# Eleventy version of the ZIP site

This project was generated by converting the provided ZIP mirror into an Eleventy (11ty) project.

## Run

```bash
npm install
npm run dev
```

Output is generated into `public/`.

## Notes

- Pages are kept as full-document Nunjucks templates (`.njk`) with front matter + permalink.
- All original assets (`assets/`, `media/`) and mirrored CDN folders are copied through unchanged.
